# TypeScript Notes & Examples

این ریپازیتوری جاییه که من هر چیزی که درباره TypeScript یاد می‌گیرم رو جمع می‌کنم. شامل مثال‌ها، توضیحات و نکات کاربردی می‌شه.

## محتوا
- مفاهیم پایه TypeScript (types, interfaces, enums)
- توابع و کلاس‌ها
- Generics
- ماژول‌ها و import/export
- نکات پیشرفته و بهترین شیوه‌ها
- پروژه‌های کوچک تمرینی

## نحوه استفاده
- این ریپازیتوری برای مطالعه شخصی و تمرین هست.
- می‌تونی هر فایل رو باز کنی و مثال‌ها رو تو محیط Node.js یا مرورگر اجرا کنی.
- پیشنهاد می‌کنم خودت هم تمرین کنی و تغییر بدی تا بهتر یاد بگیری.

## نصب
اگر می‌خوای فایل‌ها رو اجرا کنی:
```bash
# نصب TypeScript
npm install -g typescript

# اجرای یک فایل
